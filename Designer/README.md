# EduSphere 项目设计文档

本目录包含了EduSphere在线教育平台的完整设计文档和架构图。

## 📁 文件说明

### PlantUML 格式图表
- `01_System_Architecture.puml` - 系统架构图 (B/S架构)
- `02_Functional_Modules.puml` - 功能模块图
- `03_Database_ER.puml` - 数据库ER图
- `04_User_Roles_Permissions.puml` - 用户角色权限图
- `05_Business_Process_Sequence.puml` - 业务流程时序图
- `06_Frontend_Page_Structure.puml` - 前端页面结构图
- `07_API_Interface_Design.puml` - API接口设计图
- `08_Technical_Architecture_Layers.puml` - 技术架构分层图
- `09_Deployment_Architecture.puml` - 部署架构图

### 其他格式
- `10_Project_Summary.md` - 项目总结文档
- `11_System_Architecture_DrawIO.xml` - DrawIO格式系统架构图

## 🛠️ 如何使用

### PlantUML 图表
1. 安装 PlantUML 插件到你的IDE (VS Code, IntelliJ IDEA等)
2. 打开 `.puml` 文件
3. 使用 PlantUML 预览功能查看图表

### DrawIO 图表
1. 访问 [Draw.io](https://app.diagrams.net/)
2. 选择 "Open Existing Diagram"
3. 上传 `11_System_Architecture_DrawIO.xml` 文件
4. 即可查看和编辑图表

### 在线查看 PlantUML
1. 访问 [PlantUML Online Server](http://www.plantuml.com/plantuml/uml/)
2. 复制 `.puml` 文件内容
3. 粘贴到在线编辑器中查看

## 📊 图表内容概览

### 1. 系统架构图
展示了整个系统的技术架构，包括前端、后端、数据库等各层的关系。

### 2. 功能模块图
详细展示了系统的各个功能模块及其相互关系。

### 3. 数据库ER图
展示了数据库中所有表的结构和关系。

### 4. 用户角色权限图
展示了不同用户角色的权限分配。

### 5. 业务流程时序图
展示了关键业务流程的时序关系。

### 6. 前端页面结构图
展示了前端页面的组织结构和组件关系。

### 7. API接口设计图
展示了后端API接口的设计和分组。

### 8. 技术架构分层图
展示了技术架构的分层设计。

### 9. 部署架构图
展示了系统的部署架构和服务器配置。
